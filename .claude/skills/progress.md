---
name: progress
description: 创建或更新项目进度文件progress.md，记录当前开发进展、已修复的bug、待办事项
---

你正在开发侵权线索监控工具，需要创建或更新progress.md文件来记录当前进度。

## 任务目标
1. 读取当前的progress.md（如果存在）
2. 更新以下信息：
   - 当前开发阶段
   - 已完成的功能和bug修复
   - 正在进行的任务
   - 待办事项
   - 遇到的问题和解决方案

## progress.md文件格式
```markdown
# 侵权线索监控工具 - 开发进度

> 最后更新：2026-04-19
> 版本：v2r4

## 当前阶段
[阶段名称，如：筛选卡片功能开发]

## 已完成功能
- [x] 功能1（完成日期）
- [x] 功能2（完成日期）

## 已修复Bug
- [x] Bug描述（修复日期）
  - 问题原因
  - 解决方案

## 正在进行
- [ ] 任务1
- [ ] 任务2

## 待办事项
1. 待办1
2. 待办2

## 遇到的问题
- 问题描述
  - 原因分析
  - 解决方案
```

## 执行步骤
1. Read progress.md文件
2. 根据当前会话的进展更新内容
3. Write progress.md文件
4. 简短报告更新内容

## 注意事项
- 记录要简洁，只写关键信息
- 日期格式：YYYY-MM-DD
- 每次只更新变化的部分
- 保留历史记录，不要覆盖
