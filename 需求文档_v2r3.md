## 1. 界面简化需求

### 1.1 筛选区域简化

**保留的元素：**
- 项目名字（侵权线索列表）
- 已选数量
- 筛选项目（下拉框）
- 筛选公司（下拉框）

**删除的元素（不要单独一行显示）：**
- ❌ 销量筛选（不筛选、≥、从高到低、从低到低 + 数量输入框）
- ❌ 评价筛选（不筛选、≥、从高到低、从低到低 + 数量输入框）
- ❌ 侵权金额筛选（不筛选、≥、从高到低、从低到低 + 万元输入框）
- ❌ 侵权类型筛选（复选框：商标侵权、专利侵权、版权侵权、其他）
- ❌ 抓取时间筛选（日期选择：开始日期、结束日期）

## 2. 筛选方式改为点击列标题（卡片模式）

### 2.1 设计模式
- **点击列标题弹出筛选卡片**
- 所有筛选卡片都有遮罩层背景（灰色：rgba(0,0,0,0.3)）
- 每个筛选卡片都有"确认"和"重置"两个按钮
- **删除列标题箭头**（▼/▲）

### 2.2 支持筛选/排序的列

#### 数值型列（售价、销量、侵权金额、评价数量）
**筛选卡片内容：**
1. **区间筛选**：从XX到XX（自己打字输入）
   - 售价：从100RMB到500RMB（根据国家显示货币符号）
   - 销量：从100到1000（无货币符号）
   - 侵权金额：从10RMB到50RMB（根据国家显示货币符号）
   - 评价数量：从100到500（无货币符号）
2. **数值筛选**：
   - `≥`（大于等于）
   - `从高到低`（降序）
   - `从低到高`（升序）
3. **确认和重置按钮**

#### 平台选择
**筛选卡片内容：**
1. **平台列表**：根据国家显示对应平台
   - 中国：淘宝、拼多多、京东、1688
   - 美国/欧盟/英国/韩国/其他：亚马逊、SHEIN、Temu、其他
2. **选择方式**：单选或多选
3. **确认和重置按钮**

#### 店铺名称
**筛选卡片内容：**
1. **排序选项**：
   - `升序`
   - `降序`
2. **筛选选项**：勾选或单选特定的店铺名
3. **确认和重置按钮**

#### 抓取时间
**筛选卡片内容：**
1. **日期选择**：日历选择起始时间和结束时间
2. **手动输入**：筛选的具体时间段手动输入
3. **排序选项**：
   - `升序`
   - `降序`
4. **确认和重置按钮**

### 2.3 不支持筛选的列（只显示内容）
- 序号
- 图片
- 侵权类型
- 侵权地址
- 链接
- 抓取时间（已改为筛选列）

### 2.4 交互流程
```
用户点击列标题 → 显示遮罩层（灰色背景）→ 弹出筛选卡片 → 用户选择筛选条件 → 点击确认/重置 → 应用筛选/清除筛选 → 隐藏卡片和遮罩层
```

## 3. 表格横向滚动和列宽优化

### 3.1 列宽优化
1. **前面的列拉开距离**：序号、图片、侵权金额、平台、侵权类型、商品名称、店铺名称、侵权地址
2. **从售价开始后面滚动**：售价、销量、评价数量、链接、抓取时间
3. **确保横向滚动功能正常工作**

### 3.2 文本截断+tooltip
以下列支持文本截断（最多显示2行，多的隐藏，鼠标移动到上面再显示完整内容）：
- **侵权地址**：最多显示2行，超出部分隐藏，鼠标悬停显示完整内容
- **商品名称**：最多显示2行，超出部分隐藏，鼠标悬停显示完整内容
- **店铺名称**：最多显示2行，超出部分隐藏，鼠标悬停显示完整内容

### 3.3 CSS要求
- 使用 `text-overflow: ellipsis` 或 `line-clamp` 实现2行截断
- 鼠标悬停时使用 `title` 属性或自定义tooltip显示完整内容
- 确保横向滚动功能：`.infringement-table-wrapper { overflow-x: auto; }`

## 4. 侵权金额显示优化

### 4.1 侵权金额计算
- 侵权金额 = 售价 × 销量
- 根据项目国家的货币符号显示
- 数值除以10000（万的单位）

### 4.2 侵权金额列标题
- 列标题显示货币单位（万RMB、万$等）
- 根据项目国家动态更新货币单位
- 不写"万"字，显示货币符号
- 例如：侵权金额（万RMB）、侵权金额（万$）

### 4.3 侵权金额显示逻辑
- 不显示"万"字，直接显示数字和货币符号
- 例如：125.36RMB（中国）、150.23$（美国）

## 6. 界面布局结构

### 6.1 顶部区域
```
第1行：项目名字 + 已选数量
第2行：筛选项目下拉框 + 筛选公司下拉框
第3行：四个操作按钮
第4行：选择平台（点击弹出卡片，根据国家动态显示）
```

### 6.2 操作按钮
1. **+ 手动添加线索**（打开添加线索弹窗）
2. **🚀 打开搜索**（打开搜索页面）
3. **📥 导入抓取数据**（导入手动抓取的数据）
4. **📥 导出选中的侵权线索**（导出当前选中的线索）

### 6.3 表格区域
- 列数：14列
- 列标题可点击（触发排序/筛选）
- 列标题不显示箭头
- 表格支持横向滚动
- 文本截断+tooltip（商品名称、店铺名称、侵权地址）
